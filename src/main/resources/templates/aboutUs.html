About The

Project

Health Points Fitness is a full-stack application created by four student veterans meant to incentivize fitness and facilitate connections of like-minded individuals. The coding languages utilized in this application include Java, Javascript, HTML, and CSS. Frameworks include Bootstrap css and Thymeleaf. API Ninja's Exercise API is utilized during the creation of challenges. The deployment of the site was handled using Dokku.
